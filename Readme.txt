Java版本 : JavaSE-11
所用套件 ： SceneBuilder, JavaFX, FileI/O, CSS
操作方法:
1.) 登入or創建帳號
2.) 開始交易, 更換股票數據, 選擇手動/自動更新, 是否顯示指標, 圖表放大縮小
3.) 登出

匯入的數據檔案為 Excel .csv 格式
每"欄"內容依序為
1.收盤價
2.成交量
3.十日平均線
4.三十日平均線
5.RSI五日
6.RSI十日
7.MACD
8.指標線
